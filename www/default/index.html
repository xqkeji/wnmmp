什么是 WNMMP？

WNMMP 是一个专为 Windows 系统设计的绿色集成开发环境，名称来源于其包含的核心组件： Windows、Nginx、 MySQL、MongoDB 和 PHP。

WNMMP 旨在为开发者提供一个开箱即用的 Web 开发环境，无需复杂的配置过程， 解压后即可使用，通过点击install.bat自动去官网下载nginx、mongodb、mysql、PHP等开源组件并自动完成配置，通过点击start.bat启动各个组件的服务、点击stop.bat关闭组件服务，完美支持 PHP 开发的各种 Web 应用程序和网站。

无论是个人开发者构建小型应用，还是团队协作开发大型项目， WNMMP 都能满足您的需求，让您专注于代码编写而非环境配置。
